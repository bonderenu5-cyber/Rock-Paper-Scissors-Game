# Rock-Paper-Scissors-Game

Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. This project allows users to play against the computer with a clean and responsive user interface.

🚀 Features
Interactive gameplay
Responsive design for different screen sizes
Random computer choices
Instant result display (Win, Lose, or Draw)
Score tracking
User-friendly interface
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)

🎮 How to Play
Open the game in your browser.
Choose Rock, Paper, or Scissors.
The computer will randomly make its choice.
The winner is determined according to the rules:
Rock beats Scissors
Scissors beats Paper
Paper beats Rock
The result and scores are displayed instantly.
